# Wavess 2.0 : GTM Intelligence Platform (Task 2)

## Project Overview
This project is a lightweight GTM (Go-To-Market) intelligence prototype.
The goal is to showcase how external market signals from a fintech compay can be collected from public sources, structured and translated into GTM insights.

## Chosen Fintech Company - Revolut
Revolut is a global fintech company that offers online banking across multiple regions.

It is a good choice for a GTM Intelligence Platform because:
- There is a lot of **publically available** external signals.
- The company is constantly expanding into new countries.
- It operates in both consuper and enterprise segments

## What is GTM Intelligence?
In the case of this project GTM Intelligence means using external signals for decisios on:
- Ideal Customer Profile (ICP)
- Market timing
- Product-led vs. Sales-led
- Compliance and differentiation
